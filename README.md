# Peter Tadiwanashe Kawudzu — Portfolio

Personal portfolio website showcasing my work as a BSc Honours
Applied Statistics student at the University of Zimbabwe.

🔗 **Live site:** [peter-cawoodsgit.github.io](https://peter-cawoodsgit.github.io/)

## About

This site highlights my background in statistical theory,
probability, and applied data science, along with projects spanning
acceptance sampling, inventory analytics, financial econometrics,
and business intelligence reporting — built using R, Python, SQL,
SPSS, and Excel.

## Sections

- **About** — background and areas of focus
- **Skills** — technical toolkit (Python, R, SPSS, Minitab, SQL,
  Power BI, Excel)
- **Education** — academic timeline and coursework
- **Certificates** — DataCamp certifications with credential links
- **Projects** — end-to-end data analysis projects with write-ups
  and GitHub links
- **Contact** — get in touch for internships and collaborations

## Built With

Plain HTML, CSS, and JavaScript — no build step required. Uses
Typed.js, Vanilla-Tilt.js, and ScrollReveal for interactive effects,
and EmailJS for the contact form.
